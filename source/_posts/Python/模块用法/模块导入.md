## 常规导入

```python
>>> import random
>>> random.randrange(7)	# 调用相关函数时需要说明模块名
5
```

这种不消多说，只需要使用`import`一词，然后指定希望导入的模块或包即可。不过调用相关函数时需要像上面那样说明模块名。

### Python 风格

虽然可以一次性导入多个包或模块，但 Python风格指南建议将每个导入语句单独成行。

### 对模块重命名

```python
>>> import random as r
>>> r.random()
0.4967213766468448
```
