## 启用别名

### Oh My Zsh

如果你使用 `Oh My Zsh`，事情非常美好。 `Oh My Zsh` 中默认启用一个 Git 插件[^3]，提供了一套系统别名（alias）。

这样，在命令行中输入 `gaa` 就相当于 `git add --all`，快捷很多；而且这套别名应该算是国际通用的。

此外，还推荐一个可以 < 提醒当前输入命令 的 alias > 的插件：`alias-tips`[^4]
