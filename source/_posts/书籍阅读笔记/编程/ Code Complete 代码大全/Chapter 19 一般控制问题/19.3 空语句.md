在 C++等语言中，空语句即一条仅含有分号的语句

## 指导原则

### 使用空语句时应突出强调

* 让空语句的分号自占一行，并且加以缩进
* 使用一组`{ }`来强调这一空语句

这样可以告诉代码阅读者，你正在使用空语句，而不是漏写了什么

```C
while(True) {}

while(True) {
  ;
}
```

### 为空语句创建一个 DoNothing() 预处理宏或者内联函数

`toNote`
